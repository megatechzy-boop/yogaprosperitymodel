# Yoga Prosperity Model

SEO-focused PHP website for Yoga Prosperity Model.

## Local preview

```powershell
php -S 127.0.0.1:8000
```

Open `http://127.0.0.1:8000/` in a browser.

## Production requirements

- PHP 8.2 or later
- Apache with `mod_rewrite` enabled
- HTTPS
- The domain document root pointed at this directory

Razorpay and customer storage are intentionally deferred until the public website and content are approved.
